分析グループ（軸）フィルター

日記に @AI や @高配当 などのハッシュタグを書くと、 そのタグがどの「分析グループ」に属するかに応じてグラフのハブノードが色分けされます。 チェックを外したグループのタグはグラフに表示されません。

テーマ 表示中

なぜ今この銘柄が注目されるかの投資テーマ（最もよく使う）

例: @AI / @半導体 / @防衛 / @脱炭素 / @ヘルスケア / @DX / @インバウンド

マクロ感応 非表示

金利・為替・景気サイクルへの感応度を表すタグ

例: @金利感応 / @円安メリット / @景気敏感 / @インフレ / @ディフェンシブ

ビジネスモデル 非表示

稼ぎ方や成長ロジックの特徴を表すタグ

例: @M&A成長 / @プラットフォーム / @ストック収益 / @海外展開 / @ニッチトップ

資本政策 非表示

配当・株主還元・ROEへの視点を表すタグ

例: @高配当 / @累進配当 / @連続増配 / @高ROE / @株主還元強化

リスク 非表示

保有するリスクの種類を記録するタグ

例: @地政学リスク / @規制リスク / @需給変化

イベント 非表示

決算や一過性のイベントを記録するタグ（銘柄間の接続には使わない）

例: @決算注目 / @業績上方修正 / @増配 / @構造改革

色分けモード

銘柄のノードを何で色分けするか選びます。

分析グループ 推奨
@タグのグループで色分け。テーマ=紫、マクロ=黄 など
保有ステータス
保有中=緑 / 売却済み=赤 / メモ=グレー
業種
同じ業種の銘柄を同じ色で表示
実現損益
実現益=緑 / 実現損=赤（取引未確定はグレー）
クラスタ面表示

タグ・業種ハブに繋がる銘柄グループを薄い色の面で囲みます。 「どのテーマに銘柄が偏っているか」が点と線より直感的に分かります。

クラスタ面（ハル） 非表示

2銘柄以上が繋がるハブごとに、メンバー銘柄を囲む面を描画

日記関連グラフの使い方

日記・タグ・業種のつながりを「自分の投資思考の地図」として俯瞰するページです。 記録（日記・@タグ・継続記録・関連付け）が増えるほど地図が育っていきます。

ノード（形と色）の見かた
円 = 銘柄（日記）。緑=保有中 / 赤=売却済み / グレー=メモ。大きいほど接続が多い銘柄です。
六角形 = タグ・@タグ。色は分析グループ（テーマ=紫・マクロ=オレンジ など。「設定」で絞り込み可）。
角丸四角 = 業種。同じ業種の銘柄を束ねます。
水色のリング = 直近14日以内に継続記録を書いた「考えが新しい」銘柄。
薄く表示 = 90日以上 記録を更新していない保有銘柄。振り返りのきっかけにしてください。
オレンジの点 = 売却済みなのに「振り返り」ノートが未記入の銘柄。
線（つながり）の見かた
太く濃い線ほど希少で強い関連です。多くの銘柄に付く汎用的なタグの線は細く薄く表示されます。
緑の実線=追い風 / 赤の破線=向かい風。タグごとに設定した影響方向です（例: @金利上昇 が銀行には↑、REITには↓）。
黄色の破線 = コード参照。日記本文に「銘柄名(コード)」と書いた銘柄同士が自動で繋がります。
線をタップすると「なぜ繋がっているのか」の根拠（共有タグ・影響方向・言及した本文の抜粋）が見られます。
基本操作
ノードをタップ → 詳細パネル（投資理由・継続記録・タグ・関連銘柄）。
背景をドラッグで移動、ホイール / ピンチでズーム。で全体表示に戻ります。
ノードをドラッグするとその位置にピン留め。ダブルタップで解除。
パネルの「この関連のみ表示」で周辺だけに絞るフォーカス表示（深さ1〜3ホップを変更可。Esc か「解除」で戻る）。
検索ボックスに銘柄名・コードを入力すると一致ノードを強調。そのまま「一致した銘柄を中心に表示」もできます。
フィルターと設定
「表示」= 保有中・売却済み・メモの絞り込み / 「接続」= 手動リンク・タグ・業種・コード参照のつなぎ方の切り替え。
「設定」から分析グループ（軸）の絞り込み・ノードの色分け・クラスタ面（タグごとの銘柄群を囲む面）を変更できます。
右下の「未接続 N件」バッジ = どの接続にも繋がっていない銘柄の一覧。タグ付けの目安になります。
フィルター状態は URL に保存されるので、お気に入りの表示条件のままブックマークできます。
地図を育てるには
日記や継続記録に @タグ（例: @AI / @高配当）を書くと、同じタグの銘柄が自動で繋がります。
銘柄パネルの「この銘柄からリンクを作成」で、関連すると思う銘柄同士をグラフ上で直接繋げられます。
本文に 銘柄名(コード) 形式で他銘柄に触れると「コード参照」で自動的に繋がります。
