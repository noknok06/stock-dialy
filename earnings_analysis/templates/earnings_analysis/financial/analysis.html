理想型（トヨタ型）
営業CF: + 投資CF: - 財務CF: -

本業で稼いだ資金で投資を行い、借金も返済している理想的な状態

成長型（テスラ型）
営業CF: + 投資CF: - 財務CF: +

積極的な投資のために追加資金調達を行う成長企業パターン

成熟型（安定重視）
営業CF: + 投資CF: 0 財務CF: -

大型投資は控えて株主還元重視の成熟企業パターン

危険型（要注意）
営業CF: - 投資CF: + 財務CF: +

本業の赤字を資産売却と借入で補っている危険な状態
