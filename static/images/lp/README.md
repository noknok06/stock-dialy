# LP 用スクリーンショット

`templates/landing_page.html` に差し込まれる実画面の画像です。
ここにファイルを置くだけで自動的に表示されます（未設置の間はCSSモックがフォールバック表示されるため、表示は壊れません）。

すべて **ライトモード** 推奨（LP全体が紙トーンのため）。Retina で撮るなら表示サイズの2倍解像度が綺麗です。

| ファイル名 | 配置 | 推奨内容 | 向き / 目安サイズ |
|---|---|---|---|
| `showcase-home.png` | ヒーロー直下の横長ショーケース | ホーム（`/`）上部：「投資ノート」見出し＋継続バー＋日付スパインの記録が数件見える状態 | PC・横長 / 幅1280px・高さ720〜900px |
| `recall.png` | 「想起」セクションのスマホ枠 | ホームをスマホ幅で。**「答え合わせを待つ仮説」**ブロックと継続バーが入る上部 | スマホ・縦 / 幅 約720px（375〜390pt×2）・高さ1200〜1600px |
| `karte.png` | 「自己理解」セクションのスマホ枠 | 投資家カルテ（`/karte/`）の **2×2マトリクス＋得意/苦手** が入る上部。または知識ライブラリ（`/library/`） | スマホ・縦 / 幅 約720px・高さ1200〜1600px |

## 撮り方のヒント
- スマホ枠（`recall.png` / `karte.png`）は **角丸・ベゼルはこちらで付ける**ので、ブラウザのスクショ（OSのステータスバー無し）が理想。Chrome のデベロッパーツール → デバイスツールバー（iPhone 12 Pro 等）で撮ると綺麗です。
- 横長（`showcase-home.png`）は **ブラウザのアドレスバー等は不要**（こちらで簡易ブラウザ枠を付けます）。コンテンツ領域だけを撮ってください。
- 実データが少ない場合はデモアカウントで数件記録してから撮ると見栄えします。

差し替え後は `STATIC_VERSION` を上げるとキャッシュが更新されます。
