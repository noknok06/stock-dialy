# 投資タグマスタ

Claude が日記・継続記録に付けるタグはこのリストから選ぶ。

### 既存タグで対応できる場合
**リスト外のタグを新たに作ってはいけない。**
最も近い既存タグに置き換えること。

### 既存タグで対応できない場合
明らかに対応するタグがない投資テーマが出てきた場合は、タグを新設せず、
日記・継続記録の本文末尾に以下の形式で提案を残すこと：

```
【タグ提案】@提案タグ名（理由：既存タグでは○○が表現できないため）
```

例：
```
【タグ提案】@原子力（理由：@エネルギーや@脱炭素とは異なる原子力特有の政策・リスクを表現するため）
```

提案はユーザーがマスタへの追加可否を判断する。承認されるまではタグとして使用しない。
