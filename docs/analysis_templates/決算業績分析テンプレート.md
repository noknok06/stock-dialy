## 5. 決算の「質」と「継続性」の判定

> アクションを出す前に、好悪だけでなく “実力か・続くか” を分離する

**STEP1 ― 決算の質（実力か一過性か）**

|観点                    |判定       |メモ|
|----------------------|---------|--|
|一過性 vs 構造的（特需・特益の有無）  |実力 / 一過性 |  |
|利益とCFの連動（会計利益だけ伸びてないか）|連動 / 乖離  |  |
|為替・特需を剥いだ実力ベース        |良 / 並 / 悪|  |

**STEP2 ― 継続性（傾向が続くか）**

|観点                   |判定           |メモ|
|---------------------|-------------|--|
|同傾向の継続Q数（最低2-3Qで見る）  |_Q連続         |  |
|主要KPIの方向（受注/解約率/利益率等）|改善 / 横ばい / 悪化|  |
|グロースの場合：成長率は加速か減速か   |加速 / 減速      |  |

**質×継続性の総合判定**：[ 良(実力・継続) / 良(一過性) / 悪(一時要因) / 悪(構造的) / 判断不能 ]

## 8. アクション判定（機械的に出す）

> STEP1〜2の質×継続性に、STEP3のマクロ補正を重ねて決定する

|決算の質   |継続性 |マクロ|→ アクション       |
|-------|----|---|--------------|
|良(実力)  |あり  |追い風|買い増し          |
|良(実力)  |あり  |逆風 |維持（押し目待ち）     |
|良(一過性) |なし  |—  |維持（買い増さない）    |
|悪(一時要因)|—   |—  |維持・条件付き様子見    |
|悪(構造的) |悪化継続|—  |縮小・売り         |
|判断不能   |—   |—  |追加調査（3項目・期限付き）|

**投資判断のアップデート**

|項目        |Before  |After            |
|----------|--------|-----------------|
|投資判断      |買い/中立/売り|買い増し/維持/様子見/縮小/売り|
|確信度       |高/中/低   |高/中/低            |
|目標株価・想定レンジ|        |                 |

**判断根拠**（上表のどの行に該当したか。100字程度）

**アクション**

- [ ] 買い増し　[ ] 維持（押し目待ち）　[ ] 維持（一過性につき静観）
- [ ] 一部利確　[ ] 縮小・損切り　[ ] 追加調査のみ

> ※「様子見」は条件・期限を必ず付す。「追加調査」は調査3項目・期限・判断への影響をセットで書く。
