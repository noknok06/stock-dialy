# 株式分析テンプレート集

Claude（claude.ai の「株式分析テンプレート」プロジェクト）で株式分析を行う際に使うプロンプト／成果物テンプレートを一箇所にまとめたもの。

分析の役割設定・出力ルールは [`_手順.md`](_手順.md)（システムプロンプト）に集約。
タグは必ず [`tag_master.md`](tag_master.md) から選ぶ。

## テンプレート一覧

| ファイル | 用途 | 取り込み状況 |
|----------|------|--------------|
| [`_手順.md`](_手順.md) | アナリスト役割設定・出力制限・タグ運用ルール（共通システムプロンプト） | ✅ |
| [`企業説明テンプレート.md`](企業説明テンプレート.md) | ビジネスモデル・稼ぐ力・競合の俯瞰説明 | ✅ |
| [`決算業績分析テンプレート.md`](決算業績分析テンプレート.md) | 決算の質×継続性×マクロ補正でアクション判定 | ✅ |
| [`tag_master.md`](tag_master.md) | 投資タグマスタ（Claudeが選べるタグの正本） | ✅ |
| `仮説テンプレート.md` | 投資仮説の立案 | ⛔ 本文未取り込み |
| `同業ピア比較.md` | 同業ピアとの定性比較 | ⛔ 本文未取り込み |
| `テーマ関連銘柄バスケット分析.md` | テーマ／関連銘柄バスケット分析 | ⛔ 本文未取り込み |
| `外部環境からの銘柄将来分析.md` | 外部環境（マクロ・業界構造）からの将来分析 | ⛔ 本文未取り込み |
| `ニューストピック投資分析.md` | ニューストピックを起点にした投資分析 | ⛔ 本文未取り込み |

## メモ

- `tag_master.md` はこのフォルダのものが正本（唯一の所在）。旧 `docs/tag_master.md`（143行）は削除済み。`tags/management/commands/seed_master_tags.py` 等のdocstringが言及する「tag_master」はこのファイルを指す。
- 「⛔ 本文未取り込み」の5点は本文が手元に揃い次第このフォルダに追加する。
